## 📌 專案目的

兔年行大運 🐰 隨機獲取三種語言(拼音、繁體中文、英文)的兔年吉祥話，以趣味的方式解決新年吉祥話的需求

## 🎯 專案功能

- 點擊“Good Luck 兔 You"按鈕可隨機獲取兔年吉祥話
- 根據裝置的畫面尺寸變動，呈現不同的背景圖

## ✔️ 開始使用

1. 將專案 clone 到本地
2. 在本地開啟之後，透過終端機進入資料夾
3. 打開瀏覽器，並輸入 http://127.0.0.1:5500/ 進入首頁
